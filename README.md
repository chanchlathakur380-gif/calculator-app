Calculator Web Application
📌 Description

This is a simple and responsive calculator web application built using HTML, CSS, and JavaScript.
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.
The project was created to practice JavaScript logic, DOM manipulation, and responsive UI design.

🚀 Features

✔ Perform basic arithmetic operations
✔ Real-time input and instant result display
✔ User-friendly interface
✔ Responsive design for mobile and desktop
✔ Error handling for invalid operations
✔ Clear button to reset calculations

🛠 Technologies Used

HTML

CSS

JavaScript
